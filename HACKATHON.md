## The five tracks

### MagicBlock — "Solana Blitz v6"
- **Who:** MagicBlock builds Ephemeral Rollups (ERs) — on-demand SVM runtimes on Solana:
  10–50ms latency, gasless, no bridge, full base-layer composability. Plugins: Private ERs
  (TEE), VRF randomness, automation. docs.magicblock.gg
- **Prizes:** $500 (reserved: best MOBILE build) / $250 / $150 / $100 "Wizardio's Choice".
- **Eligibility gate:** project must use Ephemeral Rollups or Private ERs.
- **Judging:** creativity, technical depth, how compellingly it showcases real-time Solana.
- **Submission:** Luma — "Submission: Solana Blitz v6" event in the MagicBlock calendar
  (hackathon.magicblock.app). Needs GitHub repo + demo video OR live app link. Deadline Sunday.
- **Beyond the weekend:** MagicBlock Forge (rolling admission, engineering support).
- **Our hook:** the cheer board (`cheerboard/`) — gasless real-time tally on an ER, no wallet
  needed, 366 cheers/sec measured locally.

- **Our angle:** a real consumer product on Solana with live on-chain usage during the event.

## The meta-criterion (all five tracks share it)

Every judge here runs an investment funnel, not a prize table (Forge / Earn+grants /
pre-seed / RFS / residency). What wins across all of them: **a project that is obviously a
company, demonstrably live, with a team that will still be building it on Monday.**
